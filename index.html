My Project

Forum page

I developed a complete forum web application that enables users to register, log in, post questions, and engage in threaded discussions. The front end is built using HTML, CSS, and JavaScript, while the back end is powered by PHP with MySQLi for database operations.

Key features include:

  • User registration and login with session handling
  • Posting and replying to discussion threads
  • Input validation and secure database interaction
  • Organized layout with a responsive design

This project helped me understand full-stack development, including database design, server-side logic, and UI/UX structuring. It's a great example of how I combine frontend design with backend logic to create dynamic web applications.
